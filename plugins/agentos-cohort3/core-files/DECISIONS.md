# DECISIONS — AgentOS

> This file records durable decisions about your AgentOS and your work.

This is your own persistent record. It is a plain file you own. It travels with your AgentOS to any tool that can read it, so the decisions you capture here are remembered beyond any single conversation or product.

Use this file for decisions that should guide future work. Do not turn it into a running task list.

Read it only when making a decision, checking a prior decision, or looking up what was decided before.

To add a decision, just say "remember this decision" and it will be recorded here in the format below.

## Format

```markdown
## YYYY-MM-DD — Decision title

Decision:

What was decided.

Reason:

Why this decision was made.

Implication:

What should happen or be true going forward because of this decision.
```
