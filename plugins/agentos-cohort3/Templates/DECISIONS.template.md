# DECISIONS - <System Or Agent Name>

> This file records durable decisions that should guide future work.

Use this template by default for root or standalone AgentOS/system scaffolds.

For scoped agents inside this AgentOS, create local `DECISIONS.md` only when there is an explicit, justified local decision log need. Inheritance does not imply a local copy.

Read this file only when making decisions, checking prior decisions, or looking up system memory.

Do not turn this file into a running task log.

## Format

```markdown
## YYYY-MM-DD - Decision title

Decision:

Reason:

Implication:
```
