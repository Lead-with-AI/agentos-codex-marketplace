# Logs

This folder stores dated operating notes, maintenance notes, and verification records for `<Agent Name>` when those records need to remain available later.

Do not create logs for routine chat summaries or raw source dumps.

Use absolute dates in filenames:

- `YYYY-MM-DD-<short-topic>.md`

## Verification

Before adding a log:

- The note is worth preserving.
- Sensitive raw content is not included.
- The filename includes an absolute date.
