# Memory

This folder stores curated durable memory for `<Agent Name>` only when the memory should guide future work.

Do not use this folder as a scratchpad, transcript dump, or raw sensitive-data store.

## Allowed Memory

- <Stable fact, preference, decision, or convention this agent needs later>

## Not Allowed

- Raw private data.
- Full message threads.
- Temporary task notes.
- Facts that have not been verified.

## Verification

Before adding memory:

- The fact is durable.
- The fact belongs to this scoped agent.
- Sensitive raw content is not being stored.
- The user explicitly asked for the memory to be kept when required.
